Como usar os filtros

Tags

Adicione tags para encontrar colaboradores que as possuem. Cada tag representa uma habilidade ou característica cadastrada no sistema.

Operadores

E Colaboradores que possuem ambas as tags adjacentes.
OU Colaboradores que possuem pelo menos uma das tags adjacentes.
NÃO POSSUI Exclui colaboradores que possuem a tag seguinte.

Filtros Adicionais

Supervisor: filtra colaboradores subordinados a qualquer um dos supervisores selecionados, independentemente do tipo de supervisão.

Admitido após / Admitido até: delimita o resultado por data de admissão. Pode ser usado em conjunto com os outros filtros.

Exemplo
Angular E Java NÃO POSSUI Estagiário

Colaboradores com Angular E Java, que não sejam estagiários.
